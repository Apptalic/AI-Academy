## Overview  
This repository contains all projects aimed at teaching and enhancing your skills in building and understanding AI agentic workflows. Each project has requirements.txt to help you get started on the projects. 

You should also follow the course [here](https://apptalicacademy.com/courses/become-ai-agent-developer/) for step-by-step implementation of the projects. 




## Projects  
The repository is organized into multiple project folders. Each folder contains:  
- A `requirements.txt` file with required packages and libraries. 
- Source code and resources needed for the project.    

Explore the projects by navigating to their respective folders.  

## Getting Started  
1. Clone the repository:  
  ```bash  
  git clone https://github.com/Apptalic/AI-Academy  
  ```  
2. Navigate to the repository folder:  
  ```bash  
  cd AI-Academy 
  ```  

## Installing Dependencies  
Each project may have its own dependencies. To install them:  
1. Navigate to the project folder:  
  ```bash  
  cd project-folder-name  
  ```  
2. Install the required libraries:   
    ```bash  
    pip install -r requirements.txt  
    ```
